Unity Entities Component System sample project for uNode DOTS.

Requirements:
- Unity 6.3+
- [uNode 3.2.10+](https://assetstore.unity.com/packages/tools/visual-scripting/unode-3-pro-300160)
- [uNode DOTS add-ons](https://github.com/wahidrachmawan/uNode3-DOTS)

After opening project, you must install uNode (free/pro) then install uNode DOTS add-ons.
